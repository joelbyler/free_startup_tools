Project Management

As developers we always think we have a handle on how the project is going, but it never hurts to have a little extra help

Trello

Trello.com

Trello is the free, flexible, and visual way to organize anything with anyone.

Uses cards in columns with labels, notes, etc. to keep track of things.

Campfire

campfirenow.com

Team collaboration with real time chat.
